# 🚀 Smart Healthcare Chatbot for Disease Prediction

AI-powered healthcare chatbot that predicts diseases from user symptoms using **RAG + LLMs**. Built with **Spring Boot, Spring AI, and Ollama**, it delivers accurate, context-aware health insights through a conversational interface.

---

## 🧠 Overview

This project is an intelligent healthcare assistant that helps users understand their symptoms and get **preliminary health guidance**. It combines **Retrieval-Augmented Generation (RAG)** with modern AI models to provide reliable and explainable responses.

> ⚠️ Note: This system is for guidance only and not a replacement for professional medical advice.

---

## ✨ Key Features

- 💬 Chat-based symptom analysis  
- 🧠 AI-powered disease prediction  
- 🔍 RAG-based semantic search for accuracy  
- ⚡ Real-time response generation  
- 🧾 Context-aware medical suggestions  
- 🖥️ Clean and user-friendly UI  

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot  
- **AI Integration:** Spring AI, Ollama (LLM)  
- **Architecture:** RAG (Retrieval-Augmented Generation)  
- **Database:** H2 + Vector Store  
- **Frontend:** HTML, CSS, JavaScript  
- **Tools:** Postman, Git, IntelliJ

## ⚙️ How It Works

1. User enters symptoms in chat  
2. Backend processes input using NLP  
3. Relevant data is retrieved from vector database  
4. AI model (Ollama) generates response  
5. User receives disease prediction + advice

## 🚀 Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/smart-healthcare-chatbot.git
```

### 2️⃣ Open Project
- Import into IntelliJ / Eclipse  
- Run Spring Boot application  

### 3️⃣ Start Ollama
Make sure Ollama is running locally:
```bash
ollama run llama3
```

### 4️⃣ Run the Application
- Open browser → http://localhost:8080
